# flask を使用したwebアプリの開発

## デモ
[Pythonanywhere](https://naokey.pythonanywhere.com/)

## 仮想環境の構築

新しい環境の作成
```
python -m venv venv
```

仮想環境のアクティブ化
```
venv\Scripts\activate
```

仮想環境の非アクティブ化
```
deactivate
```
